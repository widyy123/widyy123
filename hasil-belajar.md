# Hasil Pembelajaran Git & GitHub
Pada Senin, 15 Desember 2025, saya diberikan materi mengenai Git dan GitHub, dimana isinya sebagai berikut:

1. git init : Buat repo baru di folder lokal.

git clone <url> : Copy project dari GitHub ke lokal.

2. Siklus Simpan (Wajib Hafal)

git status : Cek file yang berubah/belum disimpan.

git add . : Masukkan semua perubahan ke antrean (Staging).

git commit -m "pesan" : Simpan permanen dengan catatan.

3. Remote (GitHub/Cloud)

git remote add origin <url> : Hubungkan lokal ke repo online.

git push origin main : Kirim hasil commit ke online.

git pull : Ambil & update perubahan terbaru dari online.

4. Branch (Fitur/Eksperimen)

git branch : Cek daftar cabang.

git checkout -b <nama> : Buat & pindah ke cabang baru.

git merge <nama> : Gabungkan cabang ke branch utama.
